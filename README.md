# Portfólio Pessoal — Daniel Cardoso Martins

Bem-vindo ao repositório do meu portfólio pessoal! Este é um espaço projetado para apresentar minha trajetória acadêmica, minhas habilidades técnicas e os principais projetos que desenvolvi ao longo da minha jornada na Engenharia de Software.

**Confira o site online:** [Visualizar Portfólio](https://dcarm3.github.io/) *(Substitua pelo link real do seu GitHub Pages)*

---

## Sobre o Projeto

O design foi desenvolvido seguindo uma estética minimalista, moderna e escura (*dark mode*), inspirada em portfólios de desenvolvedores *Full-Cycle*. Ele conta com micro-interações, efeitos de ruído digital (*noise*), animações de revelação suave na rolagem (*reveal scroll*) e um cursor customizado dinâmico.

### Seções do Site:
1. **Hero:** Apresentação de impacto com animações fluidas.
2. **Sobre Mim:** Detalhes acadêmicos e foco de carreira.
3. **Habilidades:** Grade responsiva dividindo as stacks técnicas (Backend, Frontend, Banco de Dados e Ferramentas).
4. **Projetos:** Lista de trabalhos em destaque, com links diretos para os repositórios.
5. **Contato:** Formulário funcional e links para redes profissionais.

---

## Tecnologias Utilizadas

Para garantir performance máxima e controle total da estrutura e estilização, o projeto foi construído puramente com:

* **HTML5:** Estruturação semântica e acessível.
* **CSS3:** Estilização avançada utilizando CSS Variables, CSS Grid, Flexbox e animações nativas (`@keyframes`).
* **JavaScript (Vanilla):** Lógica para o cursor customizado, comportamento do menu mobile e manipulação de classes na rolagem da página.

---

## Responsividade e UI/UX

O projeto foi construído utilizando a metodologia *Mobile-First* adaptativa, garantindo uma experiência fluida em qualquer dispositivo:
* **Desktops/Ultra-wide:** Elementos maximizados e alinhados simetricamente usando CSS Grid.
* **Tablets/Mobile:** Ocultação automática de menus extensos e ativação do menu estilo hambúrguer responsivo para evitar quebras de layout.
